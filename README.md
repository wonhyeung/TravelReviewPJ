## 프로젝트 목적 
- 사용자들이 여행 경험을 기록하여 그들의 추억을 보존
## 프로젝트 요약
- 다녀온 여행지를 기록·수정·공유할 수 있음
- 랜덤으로 여행지를 추천하여 후에 갈 여행지 저장 가능
